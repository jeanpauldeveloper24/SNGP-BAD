# SNGCFP-project
Ce dépôt est l’espace officiel du Système National de Gestion des Projets BAD (SNGP-BAD) en Côte d’Ivoire, financé par la BAD. Il centralise le code source, les configurations et la documentation pour une plateforme sécurisée, conforme aux normes institutionnelles et internationales de gestion des finances publiques.
