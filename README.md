# Syst-me-National-de-Gestion-Comptable-et-Financi-re-des-Projets-SNGCFP-
Ce dépôt est l’espace officiel du SNGCFP en Côte d’Ivoire, financé par la BAD. Il centralise le code source, les configurations et la documentation pour une plateforme sécurisée, conforme aux normes institutionnelles et internationales de gestion des finances publiques.
